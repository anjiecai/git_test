    list-style-type
    list-style-position
    two properties that control the item marker and item wrapping. "list-style-type" and "list-style-position". "outside" does not allow wrapping while "inside" does. Additionally, the two can be combined to become "list-style"

    configuring your own list marker

    you can configure your own list marker when the provided types aren't enough. To do that:

    1. remove the marker by setting `list-style-type` to "none"
    2. add an image in place using the `background` css
    3. choose location and padding to provide space to the left of the text

    horizontal displaying list

    another thing we can do is orient the list horizontally. The main ways to do this are the following, with props set on the list items, not the list itself.

    • setting the `display` property to `inline` or `inline-block`
      • note: `inline-block` preferred since you can set other props like vertical margin. The item marker is auto removed when we set to inline/inline-block.
    • float the list items to keep the list item marker
